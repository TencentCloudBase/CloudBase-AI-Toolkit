# 需求文档：MCP 环境创建工具

## 介绍

当前 CloudBase MCP 中已有 `queryEnv`（查询环境信息）和 `auth`（登录与环境绑定）工具，但缺少显式的**环境创建**工具。现有的环境创建能力仅隐藏在 `auth` 工具的自动流程中（`checkAndCreateFreeEnv`），无法让用户主动选择套餐、地域、资源类型等参数。用户需要通过 CLI（`tcb env create`）或控制台手动创建环境。

本需求旨在 MCP 中新增一个环境创建管理工具，让 AI agent 能够根据用户需求主动创建 CloudBase 环境，支持选择套餐、地域、资源类型等参数。

## 需求

### 需求 1 - 查询可选套餐列表

**用户故事：** 作为 AI agent，在创建环境前需要知道用户可选的套餐有哪些及其定价，以便推荐合适的套餐。

#### 验收标准

1. When 调用套餐查询接口时，系统应当返回可用套餐列表，包含套餐 ID、套餐名称、适用场景描述。
2. When 套餐查询成功时，系统应当返回套餐的定价信息（如资源配额、价格等）。
3. When 套餐查询因权限不足失败时，系统应当给出清晰的引导，告知用户如何获取套餐信息。

### 需求 2 - 创建环境

**用户故事：** 作为 AI agent，用户告知需要创建一个新的 CloudBase 环境，agent 应当能够调用 MCP 工具创建环境，并支持指定套餐、地域、环境别名等参数。

#### 验收标准

1. When 用户提供环境别名和套餐 ID 时，系统应当创建环境并返回新环境的环境 ID。
2. When 创建环境需要指定地域时，系统应当支持 `ap-shanghai`、`ap-guangzhou`、`ap-beijing` 等常见地域参数，并默认使用 `ap-shanghai`。
3. When 创建环境需要选择资源类型时，系统应当支持 `flexdb`（文档数据库）、`storage`（存储）、`function`（云函数）的组合选择，并默认启用全部三项。
4. When 创建环境时，系统应当支持指定购买时长（月），默认 1 个月。
5. When 创建环境前，系统应当先展示配置摘要并请求用户确认，经确认后才实际执行创建。

### 需求 3 - 查询创建进度

**用户故事：** 环境创建是异步操作，AI agent 需要能够轮询环境创建状态，直到环境就绪。

#### 验收标准

1. When 环境创建提交成功后，系统应当返回新环境的 ID，并告知用户需要等待环境初始化完成。
2. When 轮询环境状态时，系统应当能通过 `queryEnv(action="info")` 查询环境的当前状态（`Status` 字段）。
3. When 环境状态为正常时，系统应当告知用户环境已就绪，可以开始使用。

### 需求 4 - 销毁环境

**用户故事：** 当用户不再需要某个环境时，AI agent 应当能够安全地销毁环境，并预先告知影响范围。

#### 验收标准

1. When 用户请求销毁环境时，系统应当先通过 `queryEnv(action="info")` 查询环境详细信息，展示给用户确认。
2. When 用户确认销毁后，系统应当要求用户再次确认（`confirm` 参数），才实际执行销毁。
3. When 销毁执行时，系统应当支持强制删除选项（针对预付费隔离中的环境）。
4. When 销毁执行时，系统应当支持跳过资源检查选项。

### 需求 5 - 变更套餐

**用户故事：** 当用户需要升降配时，AI agent 应当能够查询当前套餐并变更为目标套餐。

#### 验收标准

1. When 用户请求变更套餐时，系统应当先查询当前套餐信息。
2. When 用户指定目标套餐后，系统应当展示价格变化并请求确认。
3. When 确认后，系统应当执行套餐变更操作。

### 需求 6 - 续费环境

**用户故事：** 当用户需要续费环境时，AI agent 应当能够指定续费时长并执行续费。

#### 验收标准

1. When 用户请求续费时，系统应当支持指定续费时长（月）。
2. When 续费执行前，系统应当展示续费信息并请求确认。

## 非功能性需求

1. 工具命名统一：新增的管理工具命名为 `manageEnv`，与现有的 `manageFunctions`、`manageHosting` 等管理工具命名风格一致。
2. 操作安全：所有可能产生费用的操作（创建、变更套餐、续费）必须先展示摘要并请求用户确认。
3. 错误引导：当操作因权限不足、实名认证未完成等原因失败时，应给出清晰的引导和帮助链接。
4. 与现有流程兼容：新增工具不应破坏现有的 `auth` 工具中的自动环境创建流程。
