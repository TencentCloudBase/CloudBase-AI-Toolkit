## 常用技巧

### 切换权限模式

CodeBuddy 提供多种权限模式用于控制 AI 工具的使用授权，通过快捷键可以快速切换：

| 快捷键 | 说明 |
| --- | --- |
| `Shift+Tab` (macOS/Linux) | 切换权限模式 |
| `Alt+M` (Windows) | 切换权限模式 |

循环顺序：`default` → `bypassPermissions` → `acceptEdits` → `auto`（可用时） → `plan` → `delegate`

| 模式 | 说明 |
| --- | --- |
| **default（默认）** | 根据权限规则询问工具使用确认 |
| **bypassPermissions** | 跳过所有权限检查 |
| **acceptEdits** | 自动批准文件编辑操作，其他工具仍需确认 |
| **plan** | AI 制定计划并等待批准后再执行 |

> 参考文档：[交互模式](http://codebuddy.cn/docs/cli/interactive-mode)

### 其他常用快捷键

| 快捷键 | 说明 |
| --- | --- |
| `Ctrl+O` | 切换详细输出，显示工具使用和执行信息 |
| `Ctrl+L` | 清除终端屏幕，保留对话历史 |
| `Tab` | 切换思考模式（思考开启/关闭） |
| `Option+P` (macOS) / `Alt+P` (Windows/Linux) | 快速切换模型 |

## 配置云开发 AI 模型

你可以使用云开发提供的 AI 模型来驱动 CodeBuddy，具体配置步骤如下：

1. 在 [AI 控制台](https://tcb.cloud.tencent.com/dev#/ai?tab=text-aiModel) 中开启所需模型，并获取 `Base URL` 和 `API Key`
2. 打开 CodeBuddy，点击 **打开 Agents**
3. 在对话面板大模型列表中，选择 **配置自定义模型**
4. 填写复制的 `Base URL`、`API Key` 和模型名称，保存后即可使用

> 参考文档：[CodeBuddy 配置云开发模型](https://docs.cloudbase.net/ai/ai-tools/codebuddy) | [models.json 配置指南](https://www.codebuddy.cn/docs/cli/models)

## 常见问题

**Q: 无法使用 AI ToolKit？**
A: 请确保已登录微信开发者工具，确认网络连接正常。

**Q: 如何切换权限模式？**
A: 使用 `Shift+Tab`（macOS/Linux）或 `Alt+M`（Windows）在模式之间循环切换。Windows 用户必须使用 `Alt+M`。
