# 微信开发者工具配置指南

微信云开发 AI ToolKit 帮助开发者基于微信云开发高效构建、调试和部署小程序。

![](https://7463-tcb-advanced-a656fc-1257967285.tcb.qcloud.la/imgs/wechat-ide-extension/ide-extension.png)

**例如：**

- "用云开发做一个微信小程序点餐系统" - AI 搭页面、建云数据库、写云函数并引导预览
- "做一个预约管理系统，支持用户下单和后台改状态" - AI 用 CloudBase PostgreSQL 建模、配权限并接好前后端
- "给 Web 应用加上手机号登录" - AI 启用认证方式并接入 CloudBase SDK

无需切换到云控制台，所有操作都可以用自然语言完成。

## 前置条件

在开始配置之前，请确保满足以下条件：

<details>
<summary><strong>已安装微信开发者工具 Nightly（自带 Skills / MCP）</strong></summary>

WeChat IDE Skills 与配套 MCP **随 Nightly（开发版）内置分发**，Stable 正式版不一定包含同等能力。

- 下载与更新日志：[微信开发者工具 Nightly Build](https://developers.weixin.qq.com/miniprogram/dev/devtools/nightly_backup.html)
- 安装后确认本机有 `wechatide` 命令，以及内置 skill 目录（macOS 示例）：
  `/Applications/wechatwebdevtools.app/Contents/Resources/app.asar.unpacked/miniprogram-dev-skill`
- 日常构建用于尽快修复缺陷与敏捷特性，稳定性可能不如正式版；选用 Nightly 是为启用 AI Skills / MCP 工作流。

</details>

<details>
<summary><strong>已准备好云开发环境</strong></summary>

**云开发环境：** 请参考文档 [开通云开发环境](../../quick-start/create-env)。新用户可以免费开通体验。

</details>

## WeChat IDE Skills 与 CloudBase MCP / Skills

**微信云开发 = CloudBase × 微信。** 推荐分工：

| 层 | 作用 | 何时用 |
| --- | --- | --- |
| WeChat IDE Skills（Nightly + `wechatide`） | 执行面：调试 / 预览 / 日常云 DB·云函数·云存储 | 默认首选；微信扫码登录即可 |
| CloudBase Skills | 知识面：规则与最佳实践 | `npx skills add tencentcloudbase/cloudbase-skills -y` |
| CloudBase MCP | 补全集：权限 / 数据模型 / MySQL 等进阶能力 | IDE Skills 覆盖不到时再用 |

最小上手：

```bash
# 确认 Nightly CLI
wechatide

# 会话开始检查登录（clientName 换成你的 AI 客户端名）
wechatide -c CodeBuddy -t check_devtools_status --skill-version <versionFromSkillYaml>

# 查某个工具参数（不要猜）
wechatide -c CodeBuddy -t cloud_env_list --help
```

更完整的路径与上下文说明见 CloudBase skill：`miniprogram-development` → `references/devtools-debug-preview.md`、`references/wxide-vs-cloudbase-mcp.md`。

## 常用技巧

### 切换权限模式

CodeBuddy 提供多种权限模式用于控制 AI 工具的使用授权，通过快捷键可以快速切换：

| 快捷键 | 说明 |
| --- | --- |
| `Shift+Tab` (macOS/Linux) | 切换权限模式 |
| `Alt+M` (Windows) | 切换权限模式 |

循环顺序：`default` → `bypassPermissions` → `acceptEdits` → `auto`（可用时） → `plan` → `delegate`

| 模式 | 说明 |
| --- | --- |
| **default（默认）** | 根据权限规则询问工具使用确认 |
| **bypassPermissions** | 跳过所有权限检查 |
| **acceptEdits** | 自动批准文件编辑操作，其他工具仍需确认 |
| **plan** | AI 制定计划并等待批准后再执行 |

> 参考文档：[交互模式](http://codebuddy.cn/docs/cli/interactive-mode)

### 其他常用快捷键

| 快捷键 | 说明 |
| --- | --- |
| `Ctrl+O` | 切换详细输出，显示工具使用和执行信息 |
| `Ctrl+L` | 清除终端屏幕，保留对话历史 |
| `Tab` | 切换思考模式（思考开启/关闭） |
| `Option+P` (macOS) / `Alt+P` (Windows/Linux) | 快速切换模型 |

## 配置云开发 AI 模型

你可以使用云开发提供的 AI 模型来驱动 CodeBuddy，具体配置步骤如下：

1. 在 [AI 控制台](https://tcb.cloud.tencent.com/dev#/ai?tab=text-aiModel) 中开启所需模型，并获取 `Base URL` 和 `API Key`
2. 打开 CodeBuddy，点击 **打开 Agents**
3. 在对话面板大模型列表中，选择 **配置自定义模型**
4. 填写复制的 `Base URL`、`API Key` 和模型名称，保存后即可使用

> 参考文档：[CodeBuddy 配置云开发模型](https://docs.cloudbase.net/ai/ai-tools/codebuddy) | [models.json 配置指南](https://www.codebuddy.cn/docs/cli/models)

## 常见问题

**Q: 无法使用 AI ToolKit？**
A: 请确保已安装并登录 [Nightly 版微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/nightly_backup.html)，确认网络连接正常，且本机可运行 `wechatide`。

**Q: 装了微信开发者工具，但没有 Skills / `wechatide`？**
A: Skills/MCP 随 **Nightly** 内置。请从 [Nightly 下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/nightly_backup.html)安装开发版，不要只依赖 Stable。

**Q: 如何切换权限模式？**
A: 使用 `Shift+Tab`（macOS/Linux）或 `Alt+M`（Windows）在模式之间循环切换。Windows 用户必须使用 `Alt+M`。
