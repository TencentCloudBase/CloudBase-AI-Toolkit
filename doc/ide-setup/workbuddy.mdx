# WorkBuddy 配置指南

配置后，你可以在 WorkBuddy 的 AI 对话中直接操作 CloudBase 服务。

**例如：**

- "创建用户表" - AI 自动创建数据库集合
- "部署这个函数" - AI 自动上传并配置云函数
- "部署前端到 CDN" - AI 自动上传文件并配置托管

无需切换到云控制台，所有操作都可以在 WorkBuddy 中用自然语言完成。

## 前置条件

在开始配置之前，请确保满足以下条件：

<details>
<summary><strong>已准备好Node.js 环境和云开发环境</strong></summary>

**Node.js 环境：** 确保已安装 Node.js v18.15.0 或更高版本：

```bash
node --version
```

如果未安装，请从 [Node.js 官网](https://nodejs.org/en/download) 下载安装。

**云开发环境：** 请参考文档 [开通云开发环境](https://docs.cloudbase.net/quick-start/create-env)。新用户可以免费开通体验。

</details>

## 使用内置连接器（推荐）

WorkBuddy 已内置 CloudBase MCP 连接器，可以通过对话界面一键启用：

1. 在 WorkBuddy 对话界面中，点击输入框左下角的 **连应用** 按钮
2. 在弹窗中找到 **腾讯云 CloudBase**，点击进入详情页，点击 **连接** 启用

![WorkBuddy 内置 CloudBase 连接器](https://7463-tcb-advanced-a656fc-1257967285.tcb.qcloud.la/assets/workbuddy-connector-full.png)

![CloudBase 连接器详情](https://7463-tcb-advanced-a656fc-1257967285.tcb.qcloud.la/assets/workbuddy-connector-detail.png)

启用后，即可在对话中直接使用 CloudBase 的各项能力。

## 手动配置 MCP（备选）

如果需要在项目级别或自定义配置中使用，也可以在项目根目录创建 `workbuddy.mcp.json` 文件：

```json
{
  "mcpServers": {
    "cloudbase": {
      "command": "npx",
      "args": ["@cloudbase/cloudbase-mcp@latest"],
      "env": {
        "INTEGRATION_IDE": "WorkBuddy"
      }
    }
  }
}
```
